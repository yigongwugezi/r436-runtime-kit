# EduAgent Mock 与真实能力替换路线

## 当前结论

项目当前是“流程真实 + 内容半 mock”的状态。第一阶段已经能演示完整闭环，但后续要逐步把学习路径、资源内容、诊断和画像展示替换为真实生成。

## 已经是真实能力的部分

| 模块 | 当前状态 | 说明 |
| --- | --- | --- |
| 意图识别 | 真实 | 规则 + 示例匹配 + LLM 兜底，能根据用户输入判断闲聊、画像更新、生成方案等意图 |

## 推荐替换顺序

### 第 1 步：PlannerAgent 真实生成学习路径

目标：

- 输入学生画像、课程章节、时间安排、学习目标。
- 输出阶段化学习路径。
- 每阶段包含目标、任务、预计时间、资源类型和排序理由。

完成后可以对外说明：

```text
学习路径规划已基于用户画像和课程知识库动态生成。
```

### 第 2 步：ResourceAgent 真实生成资源

目标：

- 基于课程章节和学习路径生成至少 5 类资源：
  - 课程讲义
  - 思维导图
  - 练习题
  - 拓展阅读
  - 实操案例
  - 视频脚本

完成后可以对外说明：

```text
资源库内容已由大模型结合课程知识库生成，不再只是固定模板。
```

### 第 3 步：ProfileAgent 真实生成 8 维画像

目标：

- 根据对话画像生成雷达图分数。
- 每个维度给出解释和证据。
- 明确哪些信息来自用户输入，哪些是模型推断。

完成后可以对外说明：

```text
学习画像已由对话抽取和大模型分析共同生成。
```

### 第 4 步：DiagnosisAgent 真实诊断

目标：

- 根据画像、课程章节、学习行为追踪和练习结果生成薄弱点。
- 输出优先级、原因和推荐补救资源。

完成后可以对外说明：

```text
系统可根据学生学习行为动态调整薄弱点诊断。
```

### 第 5 步：ReviewAgent 防幻觉审核

目标：

- 检查生成内容是否和课程知识库一致。
- 标记不确定内容。
- 对高风险内容给出拦截或提醒。

完成后可以对外说明：

```text
系统具备基础防幻觉和内容安全审核机制。
```
