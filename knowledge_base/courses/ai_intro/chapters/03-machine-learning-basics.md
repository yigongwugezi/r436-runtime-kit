## 什么是机器学习

机器学习是人工智能的重要分支，它让计算机从数据中学习规律，并利用这些规律对新样本进行预测或决策。

传统编程通常由人编写明确规则，而机器学习通过数据训练模型，让模型自动学习输入与输出之间的关系。

## 基本术语

- 样本：一条数据记录。
- 特征：描述样本的输入变量。
- 标签：监督学习中的目标输出。
- 模型：从输入到输出的映射函数。
- 训练：用数据调整模型参数。
- 损失函数：衡量预测结果和真实结果之间差距的函数。
- 泛化能力：模型在未见数据上的表现能力。

## 学习类型

监督学习使用带标签数据训练模型，常见任务包括分类和回归。

无监督学习使用无标签数据发现结构，常见任务包括聚类和降维。

强化学习强调智能体与环境交互，通过奖励信号学习策略。

## 训练流程

典型机器学习流程包括：

1. 收集和清洗数据。
2. 提取特征。
3. 划分训练集和测试集。
4. 选择模型。
5. 训练模型。
6. 评估模型。
7. 调参与部署。

## 易错点

- 模型在训练集上表现好，不代表泛化能力一定好。
- 分类和回归都属于监督学习，但输出形式不同。
- 无监督学习不是“没有目标”，而是没有人工标注标签。

## Python 示例

```python
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.tree import DecisionTreeClassifier

iris = load_iris()
X_train, X_test, y_train, y_test = train_test_split(
    iris.data, iris.target, test_size=0.2, random_state=42
)

model = DecisionTreeClassifier()
model.fit(X_train, y_train)

print("accuracy:", model.score(X_test, y_test))
```
