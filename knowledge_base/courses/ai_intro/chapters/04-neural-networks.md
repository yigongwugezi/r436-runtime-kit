## 神经网络是什么

神经网络是一类受生物神经系统启发的机器学习模型。它由多个神经元按层连接构成，通过调整连接权重来学习输入和输出之间的复杂关系。

一个简单的神经网络通常包含：

- 输入层
- 隐藏层
- 输出层

每个神经元会对输入进行加权求和，再经过激活函数得到输出。

## 激活函数

如果没有激活函数，多层神经网络本质上仍然只能表示线性变换。激活函数引入非线性，使模型能够拟合复杂关系。

常见激活函数包括：

- Sigmoid
- Tanh
- ReLU

ReLU 在深度学习中非常常见，形式为：

```text
ReLU(x) = max(0, x)
```

## 前向传播与反向传播

前向传播是指输入数据从输入层经过隐藏层传到输出层，得到预测结果。

反向传播根据损失函数计算误差，并将误差从输出层向前传递，用于更新各层参数。

训练神经网络的核心过程是：

1. 前向传播得到预测结果。
2. 计算损失。
3. 反向传播计算梯度。
4. 使用优化算法更新参数。
5. 重复上述过程直到模型收敛。

## 易错点

- 神经网络不是越深越好，过深可能导致训练困难或过拟合。
- 激活函数不是可有可无，它决定模型是否具备非线性表达能力。
- 训练损失下降不代表测试效果一定变好。

## Python 示例

```python
import torch
import torch.nn as nn

model = nn.Sequential(
    nn.Linear(4, 8),
    nn.ReLU(),
    nn.Linear(8, 3)
)

x = torch.randn(2, 4)
output = model(x)
print(output.shape)
```
